====== CSS ======

===== Basics =====

  .btn { }   // classes in css with .
  #btn { }   // id's with #
  .toolbar #logo :hover { }  // access inside class toolbar a object with id "logo" on hover

===== margin/padding =====

  margin 0 0 0 0;     // top right bottom left

===== font-size =====
  em    // relative to direct or nearest neighbor font size
  rem   // root em; relative to root font size

===== position =====

  position: static    // default; normal position in flow of page. Left, top, right, bottom not working
  position: relative  // as static in flow, but left, right, top, bottom working
  position: fixed     // relative to viewpoint. Always on the same place even at scrolling. positioning with left, top, right, bottom
  position: absolute  // relative to parent. Always on the same place and positioning with left, top, right, bottom
  position: sticky    // relative to scroll position. change relative to fixed
  
===== display =====

  display: flex       // All <tags> inside are next to each other instead of among each other
  display: block      // Starts at a new line and takes whole width (e.g. <p>)
  
===== transition =====
write the transition at the element you want to change and if the property changed the transition is triggered

  a { 
    width: 100px;
    transition: width 2s; 
    }
  a:hover {                 // on hover it will take 2seconds to increase from 100px to 200px
    with:200px; 
    }
